# 疑似バーガー店員体験サービス

## 詳細
プログラムが動くと注文が行われます。声を出してしっかりと確認しよう。

## 必要なソフトウェア
  * Python

  * テスト環境
	* Ubuntu20.04
  * ROS2
	* インストールに使用したスクリプト(https://github.com/ryuichiueda/ros2_setup_scripts/blob/master/setup.bash)
